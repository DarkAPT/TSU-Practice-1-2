# Практика ML/AI engineer

Система автоматической модерации текстового контента с использованием двух моделей машинного обучения для выявления токсичных сообщений и спама. Проект демонстрирует полный цикл разработки ML-системы: от подготовки данных до развертывания REST API.

## Технологический стек

- **SQLite** - легковесная база данных
- **NLTK** - обработка естественного языка
- **Pandas** - библиотека для анализа и обработки данных
- **Mathplotlib/Seaborn** - библиотеки для визуализации данных
